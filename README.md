Thanh tìm kiếm: trên cùng, placeholder "Hôm nay bạn muốn uống gì?", ô bo góc tròn.

Tab menu: gồm 3 tab Cà phê – Trà sữa – Sinh tố, tab chọn được gạch chân nâu.

Bộ lọc ngang: các nút Tất cả, Xếp hạng, Giá, Khuyến mại có icon minh họa.

Nội dung chính: vùng trống (sẽ hiển thị danh sách đồ uống bằng RecyclerView).

Bottom navigation: 3 mục Trang chủ – Lịch sử – Tài khoản, icon rõ ràng, mục chọn tô màu.
